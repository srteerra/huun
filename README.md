# Huun

## How it works

Before writing a single word, Huun builds a complete **blueprint** in four sequential AI calls:

1. **Concept** — protagonist, antagonist, world, tone, central conflict
2. **Structure** — three-act architecture with key events pinned to exact pages
3. **Scene contracts** — per-partition rules: what must appear, what stays hidden, emotional beat, how it ends
4. **Foreshadowing** — seeds planted early with payoffs calculated later

Every partition is then generated against this contract, then validated for contradictions before reaching the reader.
